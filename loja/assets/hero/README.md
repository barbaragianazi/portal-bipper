# Imagens do hero (banner do topo) — Brindes personalizados

Coloque aqui as imagens dos slides do carrossel do topo da página.

## Como nomear o arquivo

Use o `id` do slide (veja `loja/data/brindes-personalizados.json`, campo `heroSlides`):

```
novidade.jpg
kits-corporativos.jpg
resgate-rapido.jpg
```

## Formato e tamanho recomendados

- Formato: `.jpg`
- Tamanho recomendado: **560x420px** (proporção retrato/quadrada — a imagem ocupa um quadrado de 132x132px arredondado, mas envie em resolução maior para não ficar borrada)
- Peso: idealmente abaixo de 250KB

## O que acontece se eu não subir uma imagem

O slide continua exibindo o ícone ilustrativo (SVG) como hoje. A imagem substitui o ícone automaticamente assim que o arquivo correspondente existir nesta pasta.
